Audio URL

Paste an HTTPS link to the audio file you want to play (MP3, OGG, AAC, or HLS stream). Anything Chrome's Cast Default Media Receiver can play.

Examples: a personal MP3 you've uploaded to GitHub Releases, an Internet Archive public-domain lullaby, a free sleep-radio stream URL. Spotify URLs won't work here — Spotify's Cast Receiver only accepts sessions from Spotify's own app.
